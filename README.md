Jogo de Adivinhação

Um jogo interativo desenvolvido com **HTML, CSS e JavaScript**, onde o jogador precisa adivinhar um número secreto entre 1 e 100.

---

Funcionalidades

*  Sistema de tentativas limitadas
*  Dicas em tempo real (maior ou menor)
*  Sistema de pontuação
*  Ranking com armazenamento no navegador (LocalStorage)
*  Barra de progresso das tentativas
*  Histórico de palpites
*  Efeitos sonoros estilo **arcade (8-bit)**
*  Controle de volume e botão de mute
*  Animações visuais para feedback do jogador

---

Como funciona

1. O jogo gera um número aleatório entre **1 e 100**
2. O jogador tem **10 tentativas**
3. A cada erro, o jogo informa se o número é maior ou menor
4. A pontuação começa em **100 pontos** e diminui a cada tentativa
5. O jogo termina quando:

   * ✅ O jogador acerta
   * ❌ As tentativas acabam

---

Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Web Audio API (para efeitos sonoros)
* LocalStorage (para ranking)
